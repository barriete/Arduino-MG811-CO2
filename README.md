# Arduino-MG811-CO2
Arduino Project to measure CO2 concentration in the air and upload the data to cloud

# Elements
1. Arduino MKR1010 Wifi
2. MG811 CO2 Sensor
3. DHT22 Temperature & Humidity Sensor
4. Power Supply (6V)
